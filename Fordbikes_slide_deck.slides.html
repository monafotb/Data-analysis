Preliminary Wrangling¶

Investigation Overview¶

Here I'm investigating a ford bike data. it contains many variables but i choose to focus on user types that used company bikes, start date of the ride, and durations in terms of minutes and hours.

Dataset Overview¶

dataset is consist 519700 bikes trip information it contain attributes about bike stations, coordinates, duration, start date, end date and bikers type. I took the first file which contain data for six months perion from june 2017 till december 2017. I perform data wranging first they are mentioned here: 1- Start time and end time variables should be split into sepearate date and time columns for each. 2- Drop start_time, end_time columns afterward. 3- Change date type to date/time. 4- Extract month from a date column 5- Sort values based on start time column. 6- Convert duration_sec column from seconds to minutes. 7- Convert duration from minutes to Hours.
